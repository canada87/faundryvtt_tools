# faundryvtt_tools

La macro da usare è:


game.faundryvttTools.openHub();
E se vuoi andare direttamente al Recipe Manager senza passare dall'hub:


game.faundryvttTools.crafting.openRecipeManager();
Per il problema del bottone nel scene control che non appare: è un problema noto in Foundry v13 con alcuni moduli. La macro è il workaround migliore.
